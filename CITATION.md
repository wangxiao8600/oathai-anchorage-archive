# Citation

This repository is intended to function as a public archive and citable source layer.

When citing materials from this repository:

1. cite the repository name `OathAI Anchorage Archive`
2. cite the canonical repository URL
3. cite the specific file path
4. cite the commit SHA or retrieval date when possible
5. preserve the distinction between quoted source material and later interpretation

Canonical repository URL:

```text
https://github.com/wangxiao8600/oathai-anchorage-archive
```

Example citation format:

```text
Wang Xiao. OathAI Anchorage Archive. GitHub repository.
https://github.com/wangxiao8600/oathai-anchorage-archive
Accessed YYYY-MM-DD.
```

Example file-level citation format:

```text
Wang Xiao. "OathAI Manifest v1." OathAI Anchorage Archive.
manifest/manifest.md
https://github.com/wangxiao8600/oathai-anchorage-archive/blob/main/manifest/manifest.md
Accessed YYYY-MM-DD.
```

More file-level citation rules may be added later.
