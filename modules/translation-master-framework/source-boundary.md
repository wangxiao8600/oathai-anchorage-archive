# Translation Master Framework Source Boundary

Status: public boundary note

## Purpose

This file defines what the public module may expose and what remains private by default.

## Public Surface

The public module may include:

- framework identity
- high-level method positioning
- core formula
- component list
- core terminology
- path evolution
- relation to `System and Freedom`
- relation to 21-language publication
- relation to OathAI multilingual governance

## Private by Default

The following remain outside the public module unless separately reviewed:

- raw translation workspaces
- full language-specific working directories
- unpublished review logs
- full model conversations
- private source materials
- private third-party information
- unreviewed humorous fragments
- unreviewed excerpts from internal collaboration logs

## Source Relationship

Local private source materials remain in the private source vault.

The public module is a curated surface derived from reviewed source notes.

It should not become a mirror of:

```text
日志和手稿\日常对话记录（发布前周期）\06_营销文档\02=《系统和自由》出版\00=语种
```

## Safe Public Framing

Recommended framing:

```text
The Translation Master Framework is a cross-language continuity method developed during the 21-language publication project for System and Freedom.
```

It can be described as:

- method-layer evidence
- multilingual governance predecessor
- cross-language continuity framework
- human-AI translation review structure

Avoid claiming:

- certified translation quality for every language
- formal standard status
- institutional endorsement
- automatic translation product maturity
- complete publication of the underlying working archive
