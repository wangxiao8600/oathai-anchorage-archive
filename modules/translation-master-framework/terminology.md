# Translation Master Framework Terminology

Status: public terminology baseline

## Purpose

This file records core public terms for the Translation Master Framework.

It is a terminology surface, not a full glossary of all language-specific decisions.

## Core Terms

| Term | Chinese | Meaning |
|---|---|---|
| Translation Master Framework | 翻译大师框架 | A modular method for preserving cross-language structural continuity. |
| Cross-Language Continuity | 跨语言连续性 | The ability of a work to remain structurally identifiable across language worlds. |
| Chinese Source | 中文原文 / 中文母版 | The primary semantic origin of `System and Freedom`. |
| English Alignment Layer | 英语对齐层 | The English reference layer used for terminology and structural alignment. |
| Target-Language Thinking | 目标语言原生思考 | The mature method of thinking inside the target language rather than mechanically translating through another language. |
| Core Ideas | 核心思想 | The conceptual backbone that must survive localization. |
| Core Terminology | 核心术语 | Stable terms that require consistent treatment across languages. |
| Key Phrases | 关键表达 | High-weight phrases whose rhythm, force, or symbolic value should be protected. |
| Emotional Temperature | 情感温度 | The intensity and tonal pressure of a passage. |
| Cultural Adaptation | 文化适配 | Structured adaptation that preserves meaning while respecting target-language culture. |
| Multi-language Alignment Matrix | 多语种对齐矩阵 | A comparison structure for checking how languages preserve core meaning and terminology. |
| Resurrection Methodology | 语言重生方法 | The framework's stronger formulation: a language version should become a living expression, not a mechanical copy. |
| Protocol Native Thinking | 协议原生思考 | A method mode where translation follows the work's internal protocol structure, not only sentence-level equivalence. |

## Core Formula

```text
Read Chinese, check English terms, think in the target language.
```

Chinese:

```text
读中文原文，检查英文术语，在目标语言中原生思考。
```

## Naming Boundary

Use `Translation Master Framework` for the public module name.

Use `翻译大师框架` in Chinese contexts.

Avoid presenting the framework as a certified translation standard or automated translation product.
