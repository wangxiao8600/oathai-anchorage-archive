# Publication Policy

This repository is a curated public archive.

## Core Rule

This repository should grow by selection, review, and structural placement.

It should not grow by bulk export from a private working directory.

## Material Added Here Should Be

- intentionally selected
- reviewed for privacy and scope
- stable enough for public citation
- aligned with the archive identity of this repository

## Preferred Public Material Types

- framing pages such as manifest, timeline, and layer map
- reviewed glossary or terminology files
- anchor declarations and trace files
- selected historical modules
- curated fragments with context
- public whitepapers or release artifacts

## Material That Should Usually Stay Out

- raw internal debris
- redundant drafts with no public archival value
- deployment-only material
- low-context log dumps
- material that cannot yet be responsibly framed

## Review Test

Before adding a file here, the working test is:

```text
Would this file still make sense to a future human or AI reader
if separated from the private workspace that produced it?
```

If the answer is no, it is probably not ready for this repository.
